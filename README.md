# Экспертная система по выбору языка программирования

## Описание проекта

Приложение реализует экспертную систему для подбора оптимального языка программирования: оно учитывает индивидуальные предпочтения и задачи пользователя. Графический интерфейс создан на базе PyQt6; в его основе — интерактивный опрос, а интерпретация ответов происходит за счёт механизма вывода, опирающегося на встроенную базу знаний.

## Технологии

- **Python 3**
- **PyQt6** для создания графического интерфейса
- **QMovie** для отображения анимированного фона (GIF)

## Требования

- Python 3.8 или выше
- Установленные библиотеки из `requirements.txt`

## Установка и запуск

1. Клонируйте репозиторий или скачайте исходный код:
   ```bash
   git clone https://github.com/wtorkanokra/expert-system.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd expert-system
   ```
3. Создайте и активируйте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv .venv
   ```
4. Для активации виртуального окружения:
   - **Windows**:
     ```bash
     .venv\Scripts\activate
     ```
   - **Linux/MacOS**:
     ```bash
     source .venv/bin/activate
     ```
5. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```
6. Убедитесь, что виртуальное окружение активировано (если вы его используете).
7. Запустите файл `main.py`:
   ```bash
   python main.py
   ```
8. Приложение откроется в окне.

## Структура проекта

```
├── main.py
├── README.md
├── requirements.txt
├── backgound.gif
├── data
│   └── knowledge_base.py
├── src
│   ├── app.py

│   └── screens
│       ├── __init__.py
│       ├── main_window.py
│       ├── question_window.py

```

- `main.py`: Точка входа в приложение.
- `README.md`: Документация проекта.
- `requirements.txt`: Список зависимостей проекта.
- `backgound.gif`: Анимированный фон приложения.
- `data/knowledge_base.py`: База знаний с вопросами и характеристиками языков.
- `src/app.py`: Инициализация и запуск приложения.
- `src/inference_engine.py`: Механизм вывода экспертной системы.
- `src/screens/`: Модули для графического интерфейса приложения.

## Пример использования

1. Запустите приложение.
2. Нажмите кнопку "Начать опрос" в главном окне.
3. Пройдите опрос, отвечая на вопросы о ваших предпочтениях и указывая важность различных аспектов.
4. Получите рекомендации: После завершения опроса приложение покажет вам список рекомендуемых языков программирования с процентом соответствия вашим предпочтениям.
5. Перезапуск опроса: Вы можете пройти опрос заново, нажав соответствующую кнопку в окне результатов.
